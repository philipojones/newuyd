# United Youth Developers (UYD) Website Documentation

## Project Overview

This project transforms a college website template into a comprehensive platform for United Youth Developers (UYD), a youth empowerment organization based in Uganda.

## Brand Identity Implementation

### Color Scheme

- **Primary Blue (#1e3a8a)**: Trust, stability, professionalism
- **Primary Green (#059669)**: Growth, sustainability, environmental focus
- **Gold/Yellow (#f59e0b)**: Energy, empowerment, optimism
- **Supporting Colors**: Light blue (#3b82f6), light green (#10b981), light gold (#fbbf24)

### Typography

- **Headings**: Raleway (professional, modern)
- **Body Text**: Roboto (readable, clean)
- **Navigation**: Poppins (friendly, accessible)
